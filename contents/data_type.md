## データ型とは  

コンピュータはメモリ上にある値を元に様々な計算を行います。
メモリ上にはたくさんの 0 と 1 が存在します。
この 0 と 1 の集まりをどのように解釈するか、それが「型」です。

C言語には、
* 文字を表すための型
* 整数を表すための型
* 実数（小数・浮動小数点数）を表すための型
が用意されています。
