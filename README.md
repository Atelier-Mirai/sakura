# HTML講座 発展篇

** Middleman, Slim, Sass, Git **

## Middleman(ミドルマン)
静的サイトジェネレータ。
Webサイト構築の際、ヘッダー、フッターなどを部品化。
Slim -> html, Sass -> css への変換も担う。

## Slim(スリム)
htmlを効果的に記述する。
```<>```や</>を省略でき、簡潔に書ける。

## Sass(サス)
cssを効果的に記述できる。
入れ子ができる。

## Git(ギット)
ソースコードの管理を行う。
変更を元に戻したり、バックアップやソースコードの共用が可能。
